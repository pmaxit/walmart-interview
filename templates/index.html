Assignment

For this assignment, I created a simple flask app that can download the data through http and return file statistics in json / csv format.


API

The API will have the following endpoints:

  • >/json - This endpoint will return the data in JSON format. You will need to send a GET request to retrieve the data.
  • >/csv - This endpoint will return the data in CSV format. You will need to send a POST request to retrieve the data.

How to Use

To use the API, you will need to send a GET request to the /json endpoint to retrieve the data in JSON format. To retrieve the data in CSV format, you will need to send a POST request to the /csv endpoint.
